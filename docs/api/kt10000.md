# 주식 매수주문 (kt10000)

## 기본 정보

| 항목 | 값 |
|------|-----|
| API ID | `kt10000` |
| Method | `POST` |
| 운영 도메인 | `https://api.kiwoom.com` |
| 모의투자 도메인 | `https://mockapi.kiwoom.com` (KRX만 지원) |
| URL | `/api/dostk/ordr` |

---

## Request

### Header

| Element | 한글명 | Type | Required | Length | Description |
|---------|--------|------|----------|--------|-------------|
| api-id | TR명 | String | Y | 10 | `kt10000` |
| authorization | 접근토큰 | String | Y | 1000 | Bearer {token} 형식 |
| cont-yn | 연속조회여부 | String | N | 1 | 연속조회시 Y |
| next-key | 연속조회키 | String | N | 50 | 연속조회시 이전 응답의 next-key 값 |

### Body

| Element | 한글명 | Type | Required | Length | Description |
|---------|--------|------|----------|--------|-------------|
| dmst_stex_tp | 국내거래소구분 | String | Y | 3 | KRX, NXT, SOR |
| stk_cd | 종목코드 | String | Y | 12 | 6자리 종목코드 (예: 005930) |
| ord_qty | 주문수량 | String | Y | 12 | |
| ord_uv | 주문단가 | String | N | 12 | 지정가일 때 필수 |
| trde_tp | 매매구분 | String | Y | 2 | 아래 참조 |
| cond_uv | 조건단가 | String | N | 12 | 스톱지정가(28) 사용시 |

### 매매구분 (trde_tp) 코드표

| 코드 | 설명 |
|------|------|
| 0 | 보통 (지정가) |
| 3 | 시장가 |
| 5 | 조건부지정가 |
| 6 | 최유리지정가 |
| 7 | 최우선지정가 |
| 10 | 보통(IOC) |
| 13 | 시장가(IOC) |
| 16 | 최유리(IOC) |
| 20 | 보통(FOK) |
| 23 | 시장가(FOK) |
| 26 | 최유리(FOK) |
| 28 | 스톱지정가 |
| 29 | 중간가 |
| 30 | 중간가(IOC) |
| 31 | 중간가(FOK) |
| 61 | 장시작전시간외 |
| 62 | 시간외단일가 |
| 81 | 장마감후시간외 |

---

## Response

### Header

| Element | 한글명 | Type | Required | Length | Description |
|---------|--------|------|----------|--------|-------------|
| api-id | TR명 | String | Y | 10 | |
| cont-yn | 연속조회여부 | String | N | 1 | 다음 데이터 있으면 Y |
| next-key | 연속조회키 | String | N | 50 | 다음 조회시 사용할 키 |

### Body

| Element | 한글명 | Type | Required | Length | Description |
|---------|--------|------|----------|--------|-------------|
| ord_no | 주문번호 | String | N | 7 | 성공시 주문번호 반환 |
| dmst_stex_tp | 국내거래소구분 | String | N | 6 | |
| return_code | 결과코드 | Number | | | 0=정상 |
| return_msg | 결과메시지 | String | | | |

---

## Request Example

```json
{
    "dmst_stex_tp": "KRX",
    "stk_cd": "005930",
    "ord_qty": "1",
    "ord_uv": "",
    "trde_tp": "3",
    "cond_uv": ""
}
```

## Response Example

```json
{
    "ord_no": "00024",
    "return_code": 0,
    "return_msg": "정상적으로 처리되었습니다"
}
```

---

## 주의사항

1. **종목코드**: 6자리 숫자 (예: 삼성전자 = 005930)
2. **시장가 주문**: `trde_tp`가 3일 때 `ord_uv`는 빈 문자열
3. **지정가 주문**: `trde_tp`가 0일 때 `ord_uv`에 가격 필수
4. **KRX/NXT/SOR**:
   - KRX: 한국거래소
   - NXT: 넥스트레이드
   - SOR: 최적 주문 라우팅
