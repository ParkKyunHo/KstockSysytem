# 조건검색 목록조회 (ka10171)

## 기본 정보

| 항목 | 값 |
|------|-----|
| API ID | `ka10171` |
| Method | `WebSocket` |
| 운영 도메인 | `wss://api.kiwoom.com:10000` |
| 모의투자 도메인 | `wss://mockapi.kiwoom.com:10000` (KRX만 지원) |
| URL | `/api/dostk/websocket` |

**주의**: REST API가 아닌 WebSocket 연결 필요!

---

## 개요

영웅문4 HTS에서 생성한 조건검색식 목록을 조회합니다.

---

## Request

### Header (WebSocket 연결 시)

| Element | 한글명 | Type | Required | Description |
|---------|--------|------|----------|-------------|
| authorization | 접근토큰 | String | Y | Bearer {token} 형식 |

### Body

| Element | 한글명 | Type | Required | Description |
|---------|--------|------|----------|-------------|
| trnm | TR명 | String | Y | `CNSRLST` 고정값 |

---

## Response

### Body

| Element | 한글명 | Type | Description |
|---------|--------|------|-------------|
| return_code | 결과코드 | String | 0=정상 |
| return_msg | 결과메시지 | String | |
| trnm | 서비스명 | String | `CNSRLST` |
| data | 조건검색식 목록 | List | |
| - seq | 조건검색식 일련번호 | String | 조건식 ID (000, 001, ...) |
| - name | 조건검색식 명 | String | HTS에서 설정한 이름 |

---

## Request Example

```json
{
    "trnm": "CNSRLST"
}
```

## Response Example

```json
{
    "return_code": "0",
    "trnm": "CNSRLST",
    "data": [
        {"seq": "000", "name": "주도주매수조건"},
        {"seq": "001", "name": "단타조건"},
        {"seq": "002", "name": "급등주포착"}
    ]
}
```

## WebSocket 연결 예제

```python
import asyncio
import websockets
import json

async def get_condition_list(token):
    uri = "wss://api.kiwoom.com:10000/api/dostk/websocket"

    headers = {
        "authorization": f"Bearer {token}"
    }

    async with websockets.connect(uri, extra_headers=headers) as ws:
        # 조건식 목록 요청
        request = {"trnm": "CNSRLST"}
        await ws.send(json.dumps(request))

        # 응답 수신
        response = await ws.recv()
        data = json.loads(response)

        if data.get("return_code") == "0":
            for condition in data.get("data", []):
                print(f"[{condition['seq']}] {condition['name']}")

        return data

# 실행
# asyncio.run(get_condition_list("your_token_here"))
```

## 주의사항

1. **조건식 생성**: 영웅문4 HTS에서만 가능
2. **통신 방식**: REST(ka10172)와 WebSocket(ka10173) 둘 다 지원
   - REST 우선 사용 (폴링 방식)
   - 실시간 필요시 WebSocket 사용
3. **seq 값**: 다른 조건검색 API (ka10172, ka10173, ka10174)에서 사용
