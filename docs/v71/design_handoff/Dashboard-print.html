주식 대시보드
디자인 제안

차트 렌더링 없음. 숫자 입력과 테이블 중심. 사용자는 HTS에서 차트를 보고 판단하며, 대시보드는 그 판단을 입력받아 시스템에 전달합니다.
01
저녁 리포트
장세 요약 + 주도주 리스트. 스플릿 / 테이블 / 카드 3가지 레이아웃.
02
박스 설정
종목 → 전략 → 타임프레임 → 가격 4단계 입력. 마스터·디테일 / 그리드.
03
포지션 모니터
실시간 손익 + 청산 기록. 테이블 / 카드.
04
추적 종목 관리
상태 필터 + 이탈 경고. 테이블 / 칸반.
